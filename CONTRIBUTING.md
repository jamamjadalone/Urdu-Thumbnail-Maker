# Contributing to Urdu Thumbnail Maker

We welcome contributions! Here's how to get started:

## Getting Started

1. Fork the repository
2. Clone your fork: git clone https://github.com/yourusername/Urdu-Thumbnail-Maker.git
3. Create a feature branch: git checkout -b feature/amazing-feature

## Making Changes

- Follow existing code style
- Keep changes focused
- Add comments for complex logic
- Test thoroughly in multiple browsers

## Committing

Use clear commit messages:
`
git commit -m "Add: description of changes"
git commit -m "Fix: description of bug fix"
`

## Pushing & Pull Request

1. Push to your fork
2. Open a Pull Request with clear description
3. Reference any related issues

## Code Style

- Use meaningful variable names
- Keep functions focused
- Test on Chrome, Firefox, Safari, Edge
- Verify Urdu text renders correctly

## Questions?

Open an issue or start a discussion!
