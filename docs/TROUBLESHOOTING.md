# Troubleshooting Guide

## Text Not Displaying

**Problem:** Urdu text appears as boxes or symbols

**Solutions:**
1. Check keyboard language (set to Urdu)
   - Windows: Settings → Language → Add Urdu
   - Mac: Preferences → Keyboard → Input Sources → Add Urdu

2. Try different browser (Chrome recommended)

3. Clear browser cache:
   - Ctrl+Shift+Delete (Windows)
   - Cmd+Shift+Delete (Mac)

4. Update browser to latest version

## Fonts Not Loading

**Problem:** Text appears in default font, not Nastaleeq

**Solutions:**
1. Clear browser cache
2. Try different font from dropdown
3. Refresh page (Ctrl+F5 or Cmd+Shift+R)
4. Check browser console for errors (F12)

## Download Not Working

**Problem:** "Download" button does nothing

**Solutions:**
1. Use supported browser: Chrome, Firefox, Safari, Edge
2. Check browser download permissions
3. Try incognito/private mode
4. Check browser console (F12) for errors

## Image Upload Issues

**Problem:** Background image doesn't appear

**Solutions:**
1. Try smaller image (< 10MB)
2. Use supported formats: JPG, PNG, WebP
3. Try different image
4. Check console (F12) for errors

## Text Cut Off

**Problem:** Urdu text goes outside preview box

**Solutions:**
1. Reduce font size
2. Add background overlay for contrast
3. Change text color to brighter shade
4. Add text outline (stroke)

## Browser Crashes

**Problem:** App becomes unresponsive

**Solutions:**
1. Close other browser tabs
2. Clear browser cache
3. Update browser
4. Try different browser
5. Try simpler background images
