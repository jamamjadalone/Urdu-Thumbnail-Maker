# Setup & Installation Guide

## Prerequisites
- Any modern web browser (Chrome, Firefox, Safari, Edge)
- Git (optional)

## Installation

### Method 1: Use Live Demo (Easiest)
Simply visit the live website - no installation needed!

### Method 2: Download & Run Locally

#### Step 1: Get Files
\\\ash
git clone https://github.com/jamamjadalone/Urdu-Thumbnail-Maker.git
cd Urdu-Thumbnail-Maker
\\\

#### Step 2: Open in Browser
Double-click index.html

Or use a local server:
\\\ash
# Python 3
python -m http.server 8000

# Node.js
npx http-server
\\\

## Troubleshooting

### Fonts not loading?
- Clear browser cache (Ctrl+Shift+Delete)
- Try different browser
- Check console for errors (F12)

### Text not displaying?
- Set keyboard to Urdu
- Try different font from dropdown
- Clear cache and refresh

### Download not working?
- Use Chrome, Firefox, Safari or Edge
- Check browser permissions
- Try incognito/private mode
