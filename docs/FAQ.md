# FAQ - Frequently Asked Questions

## General

**Q: Is this completely free?**
A: Yes, 100% free with no ads or sign-up required.

**Q: Do I need to install anything?**
A: No, it works in any browser. No downloads needed.

**Q: Is my data private?**
A: Yes! Everything happens locally in your browser. No uploads to servers.

## Urdu Support

**Q: Which Urdu fonts are supported?**
A: 
- Jameel Noori Nastaleeq
- Nafees Nastaleeq
- Alvi Nastaleeq
- Mehr Nastaleeq
- System Font

**Q: Why doesn't Urdu text display correctly?**
A: 
- Check keyboard language (set to Urdu)
- Try a different font
- Clear browser cache
- Try different browser

## Features

**Q: Can I use this for non-YouTube purposes?**
A: Yes! It generates standard PNG images usable anywhere.

**Q: Can I add multiple text layers?**
A: Currently no, but it's planned for future versions.

**Q: Can I save my designs?**
A: Currently you can only download. Cloud save planned for v2.0.
